# Routing

Use this file to decide when the general skill should delegate.

## Delegate To `$research-memory-retrieval`

Delegate when the topic includes any of these themes:

- long-term memory
- agent memory
- episodic or semantic memory
- RAG
- retrieval quality
- reranking
- indexing
- provenance
- freshness
- context management
- retrieval failures

## Keep Work In `research-capture`

Stay in the general skill for repo-aware capture and broad research outside that domain, such as:

- the exact command for a file or path
- a failing test, stack trace, or local error log
- repo-specific preflight or setup blockers
- change review, reviewer concerns, or risk areas
- checked-in `.codex/repo-profile.toml` or nearest `AGENTS.md` guidance

- product comparisons
- technical literature reviews outside memory/retrieval
- sourcing evidence for non-memory architectural decisions
- historical, market, legal, or general technical research that still needs capture
